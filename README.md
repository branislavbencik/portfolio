# branislavbencik.com

Portfolio site.

**[→ branislavbencik.com](https://branislavbencik.com)**

## Stack

Next.js 16 · Tailwind CSS 4 · MDX · Motion · Geist Sans/Mono · Static export on Vercel

No SSR, no API routes, no CMS. MDX files are the CMS.

## Quality

Playwright for automated responsive checks across breakpoints. Codebase quality maintained via `STATUS.md` tracking, a supervising AI strategist (separate Claude project reviewing decisions), and periodic Claude Code `/insights` runs — surfacing best practices, evaluating benefit vs. cost, implementing what's worth it.

## Built with Claude Code

Design specs extracted from Figma, implementation driven by [Claude Code](https://docs.anthropic.com/en/docs/claude-code), reviewed and iterated by me. The `CLAUDE.md` in this repo is the instruction set Claude Code reads at session start — design tokens, component architecture, conventions.
